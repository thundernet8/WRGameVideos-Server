## WRGameVideos-Web Clent

** 因考虑到视频版权原因和 App Store 的审核问题，项目放弃 **

WRGameVideos Web Client

此项目本为一个视频聚合应用的 API 端，移动适配的 web 客户端请求数据生成页面，并通过Webview 组件加载至 iOS 应用中，少量的交互由 Javascript和 Objective-C 通信完成
